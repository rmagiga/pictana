# Pictana - プロダクト概要

ローカルストレージ上の画像を閲覧・管理するための Flutter デスクトップ/モバイルアプリケーション。

## 主な機能

- ストレージ（フォルダ）選択とパーミッション永続化
- フォルダブラウジング
- ギャラリーグリッド表示（サムネイル、インクリメンタル読み込み、無限スクロール）
- 画像ビューア（ズーム、スワイプ、隣接画像プリロード）
- ソート・ファイル名検索・MIME type フィルター
- お気に入り管理
- ダーク/ライトテーマ切り替え
- キャッシュ管理

## 対応画像形式

JPEG, PNG, WebP, GIF, HEIC, HEIF, AVIF

## ターゲットプラットフォーム

- Windows（主要ターゲット）
- Android（SAF ベースのストレージアクセス、USB OTG 対応）

## パフォーマンス目標

- 10,000枚規模の画像一覧で 60fps を維持する遅延ロード
- Isolate を用いた非同期デコード
- Virtual Scroll / Viewport 最適化

## 言語方針

- コード内コメント・ドキュメンテーションコメントは日本語
- UI テキストは日本語
- 変数名・クラス名・関数名は英語（Dart 慣例に従う）
- AI への応答・コミットメッセージも日本語
