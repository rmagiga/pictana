# Optrig - プロダクト概要

Optrig はローカルストレージ上の画像を閲覧・管理するための Flutter デスクトップ/モバイルアプリケーション。

## 主な機能

- ストレージ（フォルダ）選択とパーミッション永続化
- フォルダブラウジング
- ギャラリーグリッド表示（サムネイル、インクリメンタル読み込み、無限スクロール）
- 画像ビューア（ズーム、スワイプ、隣接画像プリロード）
- ソート・ファイル名検索・MIME type フィルター
- ダーク/ライトテーマ切り替え
- キャッシュ管理

## 対応画像形式

JPEG, PNG, WebP, GIF, HEIC, HEIF, AVIF

## ターゲットプラットフォーム

- Windows（主要ターゲット）
- Android（対応予定/対応中）

## 言語

- コード内コメント・ドキュメントは日本語で記述する
- UI テキストも日本語
